Who made this?

Originally made in 2015 by Josiah Wolf Oberholtzer.

Updated 2023 by Andy Radburn.

Discograph2 is made with a little help from these great tools and packages:

  • Python 3
  • Peewee
  • D3 and Bootstrap CSS
  • Machina-JS
  • Flask, Gunicorn and supervisor.d

Discograph2 would also be impossible without the generous public data dump that Discogs.com provides monthly.

If something is not working, please file a bug report on Github.

If any artist or label information is missing or incorrect, it can be updated on Discogs.com. It may take time, even months, for the updates to be processed in Discogs, then updated here in Discograph

You can fork Discograph2 on GitHub.
